# 项目：我的待办事项列表 (桌面小组件)

这是一个基于 Tauri、React 和 Vite 构建的轻量级、半透明、无边框桌面“小组件”风格的待办事项列表。

## 架构与技术栈
- **外壳：** Tauri (v2)
- **前端框架：** React (v19)
- **构建工具：** Vite (v8)
- **语言：** Rust (后端), JavaScript/JSX (前端)
- **图标：** Lucide React
- **样式：** 原生 CSS 结合毛玻璃特效 (backdrop-filter)
- **数据持久化：** 存储在系统 `AppData` 目录下的 JSON 文件中 (`todos.json`, `archive.json`, `settings.json`)。

## 项目结构
- `src-tauri/`: 包含 Rust 后端代码、配置和资源。
    - `src/main.rs`: 主进程逻辑，包含 IPC 命令实现、托盘管理和窗口置顶逻辑。
    - `capabilities/`: 权限配置。
- `src/`: 包含 React 前端代码。
    - `App.jsx`: 核心应用逻辑和 UI 界面。
    - `index.css`: 全局样式及毛玻璃效果定义。
    - `App.css`: 应用特定样式。
- `public/`: 静态资源文件。

## 核心功能
- **小组件 UI：** 透明无边框窗口，支持调节透明度、字体大小和高度。
- **可拖拽性：** 顶部页眉区域允许移动窗口（除非已锁定）。
- **吸附边缘：** 窗口在移动时会自动限制在屏幕工作区内，防止超出边界。
- **系统托盘：** 可最小化至托盘；右键托盘查看选项，左键恢复窗口。
- **持久化：** 自动将任务、归档和设置保存至本地 JSON。
- **任务管理：** 
    - **添加：** 通过底部输入框快速添加。
    - **详情/编辑：** 双击任务进入详情模态框，支持编辑描述和设置截止日期。
    - **状态：** 根据截止日期显示不同状态颜色（过期、紧急、近期）。
    - **上下文菜单：** 右键任务可进行置顶、完成归档或删除操作。
    - **排序：** 置顶任务优先，其次按截止日期和时间戳排序。
    - **归档：** 已完成任务进入归档页，支持按月分页加载。

## 开发命令
- `npm run dev`: 启动 Tauri 开发环境（包含前端热更新）。
- `npm run build`: 构建生产版本。
- `npm run lint`: 执行 ESLint 代码检查。

## 开发规范
- **IPC 通信：** 使用 `@tauri-apps/api/core` 中的 `invoke` 进行前后端交互。
- **样式规范：** 使用 `index.css` 中 `:root` 定义的 CSS 变量。
- **状态管理：** 使用 React 本地状态，通过 `useEffect` 与后端同步数据。
- **拖拽区域：** 通过调用 `appWindow.startDragging()` 实现自定义拖拽逻辑，避免使用原生 `-webkit-app-region` 带来的交互限制。

## 生产构建
使用 `npm run build` 生成。构建后的安装包将位于 `src-tauri/target/release/bundle/` 下。
